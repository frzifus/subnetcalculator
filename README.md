# subcalc
------
Very simple cli based subnet calculator

### Build
```
make
```
### Install
```
make install
```
### Uninstall
```
make uninstall
```
------
### Usage
**Example: calculate 3 subnets on a specific ip**
```
subcalc -ip 192.168.0.1 -n 3 -m 255.255.255.240
```
**Help**
```
subcalc -h
```
